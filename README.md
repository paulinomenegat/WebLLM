Local IA - WebLLM Local Chat 🤖

English | Português

🇺🇸 English

Description

Local IA is a privacy-first, fully browser-based Large Language Model (LLM) chat interface. It leverages WebGPU and the WebLLM engine to download and run inference models directly on the user's device. Since everything runs locally, no data is ever sent to external servers, ensuring 100% privacy and zero API costs.

Key Features

100% Local Inference: Models run directly in the browser using the user's local hardware (GPU).

Total Privacy: No cloud computing, no data logging, and no API keys required. Your conversations stay on your machine.

Multi-Model Support: Easily switch between highly optimized, quantized models including:

Llama 3 (8B)

Phi-3 Mini (3.8B)

Mistral v0.3 (7B)

Gemma (2B)

Qwen 2 (1.5B)

Multi-language UI: Interface translation available in English, Portuguese (BR), Spanish, and Chinese.

Advanced Parameter Control: Fine-tune the AI's behavior by adjusting the Temperature and Max Tokens limits in real-time.

Real-time Progress Tracking: Visual feedback for model downloading and loading status.

Streaming Responses: Watch the AI generate text in real-time, just like premium cloud-based SaaS solutions.

Technical Specifications

Core Engine: @mlc-ai/web-llm (v0.2.78)

Hardware Acceleration: WebGPU API (Requires a compatible modern browser like Chrome, Edge, or Brave).

Frontend Stack: HTML5, vanilla JavaScript (ES Modules), Tailwind CSS (via CDN) for styling, and Lucide Icons.

Model Format: MLC Quantized models (q4f16_1 / q4f32_1) optimized for low VRAM usage and high performance.

🇧🇷 Português

Descrição

O Local IA é uma interface de chat para Grandes Modelos de Linguagem (LLMs) totalmente baseada no navegador e com foco em privacidade. O sistema utiliza WebGPU e o motor WebLLM para baixar e executar a inferência dos modelos diretamente no dispositivo do usuário. Como tudo roda localmente, nenhum dado é enviado para servidores externos, garantindo 100% de privacidade e zero custo com APIs.

Principais Funcionalidades

Inferência 100% Local: Os modelos rodam diretamente no navegador utilizando o hardware (GPU) local do usuário.

Privacidade Total: Sem computação em nuvem, sem coleta de dados e sem necessidade de chaves de API. Suas conversas não saem da sua máquina.

Suporte a Múltiplos Modelos: Alterne facilmente entre modelos quantizados e altamente otimizados, incluindo:

Llama 3 (8B) - Poderoso

Phi-3 Mini (3.8B) - Rápido

Mistral v0.3 (7B)

Gemma (2B) - Leve

Qwen 2 (1.5B) - Super Leve

Interface Multilíngue: Interface traduzida para Inglês, Português (BR), Espanhol e Chinês.

Controle Avançado de Parâmetros: Ajuste o comportamento da IA alterando a Temperatura e o Limite de Tokens Máximos em tempo real.

Acompanhamento em Tempo Real: Feedback visual detalhado durante o download e carregamento dos modelos para a memória.

Respostas em Streaming: Veja a IA gerar texto em tempo real, proporcionando uma experiência fluida idêntica a soluções SaaS premium na nuvem.

Especificações Técnicas

Motor Principal: @mlc-ai/web-llm (v0.2.78)

Aceleração de Hardware: API WebGPU (Requer um navegador moderno compatível, como Chrome, Edge ou Brave).

Stack Frontend: HTML5, JavaScript Vanilla (ES Modules), Tailwind CSS (via CDN) para estilização responsiva, e Lucide Icons.

Formato dos Modelos: Modelos quantizados MLC (q4f16_1 / q4f32_1) otimizados para baixo consumo de VRAM e alta performance.
